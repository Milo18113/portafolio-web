Wordle Solver

Esta aplicación fue desarrollada en Python para poner en práctica mis habilidades con el lenguaje.

Se emplañaron dos técnicas de predicción de palabras para ganar el juego, las cuales se describen a continuación.

El sistema funciona con una interfaz de consola, y próximamente se añadirá un front-end mediante la librería streamlit de Python.
